چرا Web Container مستقیماً Docker Socket ندارد؟

Docker Socket عملاً دسترسی مدیریتی بسیار قدرتمندی به Host می‌دهد. این پنل به‌جای mount کردن socket در Web UI، از یک Agent محلی با API محدود و allowlist استفاده می‌کند.

آدرس پیشنهادی

manager.example.com → http://localhost:8788

پورت‌های Oracle

22/tcp  فقط IP مدیریت
80/tcp  سایت‌های Hestia
443/tcp سایت‌های HTTPS
8888/tcp فقط در حالت Public RDP و فقط CIDR مجاز

RDP بدون پورت ورودی

cloudflared access rdp --hostname rdp.example.com --url rdp://localhost:13389
mstsc.exe /v:localhost:13389
