BSM Community 0.2.0

یک پنل عمومی و بدون اطلاعات شخصی برای نصب، تنظیم، مانیتورینگ، لاگ‌گیری، بکاپ، بازیابی و عیب‌یابی سرور Ubuntu 24.04.

حریم خصوصی: هیچ IP، دامنه، ایمیل، Token یا رمز واقعی داخل Repository نیست. اطلاعات هنگام Bootstrap از کاربر گرفته و فقط روی سرور ذخیره می‌شود.

نصب از GitHub عمومی

git clone https://github.com/YOUR-USER/YOUR-REPOSITORY.git
cd YOUR-REPOSITORY
sudo ./bootstrap.sh

Bootstrap دامنه، Hostname پنل، Timezone، کاربر فایل‌ها، رمز اولیه و روش دسترسی را می‌پرسد.

ساده‌ترین ورود اولیه

حالت پیش‌فرض یک آدرس موقت trycloudflare.com می‌سازد و همان لحظه آن را نمایش می‌دهد. این آدرس فقط برای راه‌اندازی اولیه است. پس از ورود، Tunnel دائمی و Cloudflare Access را تنظیم کن.

Quick Tunnel برای Production نیست. پنل همچنان Login داخلی دارد، اما برای استفاده دائمی Named Tunnel و Cloudflare Access لازم است.

دستیار AI

در صفحه AI می‌توان Provider، Model و API Key را وارد کرد. کلید روی Host با مجوز root ذخیره می‌شود و به مرورگر بازگردانده نمی‌شود.

کاربر مشخص می‌کند کدام اطلاعات ارسال شوند: وضعیت سیستم، کانتینرها، تنظیمات Redacted، Command Ledger، لاگ کانتینر انتخابی یا لاگ سرویس انتخابی. قبل از ارسال، Preview دقیق نمایش داده می‌شود.

محدودیت امنیتی: AI فقط مشاوره می‌دهد. پاسخ آن هیچ‌گاه به‌صورت خودکار اجرا نمی‌شود.

Command Ledger

/var/log/bsm/command-ledger.jsonl
/var/log/bsm/command-ledger.sh

دستورها، زمان، کاربر، Exit Code، مدت اجرا، خلاصه خروجی و Hash تغییر فایل‌ها ثبت می‌شود. Secretهای شناخته‌شده قبل از ثبت حذف می‌شوند.

انتشار امن Repository

./scripts/public-release-check.sh

همچنین Secret Scanning و Push Protection را در GitHub فعال کن و فایل‌های /etc/bsm یا Logهای واقعی را Commit نکن.

مسیرهای مهم

/etc/bsm/       تنظیمات و Secretهای root-only
/var/lib/bsm/   Jobها، Diagnostics و Updateها
/var/log/bsm/   Logها و Command Ledger
